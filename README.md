# NearByBites 🍽️

**Discover Amazing Food Around You**

NearByBites is a modern AI-powered restaurant discovery and food recommendation platform that helps users find nearby restaurants, explore trending dishes, and get personalized food suggestions through an intelligent chatbot experience.

Built with a premium, responsive UI inspired by leading food-tech platforms, NearByBites combines restaurant search, location-based discovery, and conversational AI to create a seamless food exploration experience.

## ✨ Features

* 🔍 Smart Restaurant Search
* 📍 Nearby Restaurant Discovery
* 🤖 AI Food Recommendation Chatbot
* ⭐ Restaurant Ratings & Reviews
* 🍕 Trending Food Categories
* ❤️ Save Favorite Restaurants
* 🌙 Modern Dark Mode Interface
* 📱 Fully Responsive Design
* ⚡ Fast and Optimized Performance
* 🎨 Premium Animations & Interactions

## 🛠 Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Framer Motion
* Shadcn UI
* Lucide Icons

## 🚀 Key Highlights

* Beautiful startup-quality user interface
* Interactive restaurant discovery experience
* Personalized food recommendations
* Hindi foodie dialogues for enhanced engagement
* Modern glassmorphism design system
* Mobile-first responsive layout
* Deployment-ready architecture

## 🎯 Vision

NearByBites aims to simplify food discovery by helping users quickly find great restaurants, explore new cuisines, and receive intelligent dining recommendations tailored to their preferences and location.

## 📸 Preview

NearByBites brings together restaurant search, AI assistance, and food exploration into a single modern platform designed for food lovers.
